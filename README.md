# Tic Tac Toe Webanwendung (Flask)
Dies ist eine moderne Tic Tac Toe Webanwendung, entwickelt mit Python, Flask, HTML, CSS und JavaScript.
Die Anwendung unterstützt sowohl den Player-vs-Player-Modus als auch den Player-vs-Computer-Modus mit einfacher KI.
Live Demo:  
https://tic-tac-toe-webapp-flask.onrender.com
_____________________________________________________________________________________________________________________
## Funktionen
- Player vs Player Modus
- Player vs Computer Modus (KI mit Blockierlogik)
- Permanentes Scoreboard (Speicherung in JSON-Datei)
- Automatische Gewinnererkennung
- Hervorhebung der Gewinnlinie mit Animation
- Modernes und responsives Design
- Kommunikation zwischen Frontend und Backend über REST API (Fetch / JSON)
- Online Deployment mit Render

## Installation und Start
Repository klonen:
git clone https://github.com/mariia-ae/tic-tac-toe-flask.git
In Projektordner wechseln:
cd tic-tac-toe-flask
Abhängigkeiten installieren:
pip install -r requirements.txt
Anwendung starten:
python app.py
Im Browser öffnen:
http://127.0.0.1:5000
